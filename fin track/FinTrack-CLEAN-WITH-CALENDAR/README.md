## 🚀 **How to Use Calendar:**

### **1. Calendar View**
- **Navigate months** using the dropdown selectors
- **Click any date** to add transactions
- **See transaction indicators** on days with activity
- **View net amounts** for each day

### **2. Adding Transactions**
- **Click a date** to open the transaction modal
- **Select type** (Income/Expense)
- **Enter amount** and description
- **Add category** (optional)
- **Save transaction**

### **3. Viewing Transactions**
- **Click dates with indicators** to see existing transactions
- **View all transactions** for that day
- **See monthly totals** at the top
