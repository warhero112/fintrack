## 🎨 **Design System:**

### **Colors**
- **Primary**: `#111418` (Dark text)
- **Secondary**: `#617589` (Muted text)
- **Background**: `#FFFFFF` (White)
- **Cards**: `#F0F2F4` (Light gray)
- **Accent**: `#1380EC` (Blue)
- **Success**: `#078838` (Green)
- **Error**: `#E73908` (Red)

### **Typography**
- **Font Family**: Manrope, "Noto Sans", sans-serif
- **Headings**: Bold, tight tracking
- **Body**: Normal weight, readable
- **Small Text**: 13px, bold for labels

### **Spacing**
- **Consistent**: 4px base unit
- **Padding**: 16px, 24px, 32px
- **Margins**: 8px, 16px, 24px
- **Gaps**: 4px, 8px, 16px, 24px

### **Components**
- **Cards**: Rounded corners, subtle shadows
- **Buttons**: Rounded, proper sizing
- **Inputs**: Clean borders, focus states
- **Progress**: Visual progress indicators
